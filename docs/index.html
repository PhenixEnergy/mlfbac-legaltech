📐 Vektor-Repräsentation von Rechtstexten

Für jeden Begriff t in einem Textsegment S wird ein Gewicht wt,S berechnet:

Formel: wt,S = bt × (ft,S + pt,S × 0.5) × ct
Variable Beschreibung Wertebereich
bt Basisgewicht für Begriff t aus rechtlicher Domäne 2.0 - 6.0
ft,S Häufigkeit exakter Übereinstimmungen von Begriff t in Segment S ≥ 0
pt,S Häufigkeit partieller Übereinstimmungen (Fuzzy-Matching) ≥ 0
ct Kontextfaktor basierend auf Segmentposition und Umgebung 0.5 - 2.0
