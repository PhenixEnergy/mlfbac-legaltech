🏗️ Erweiterte Segmentierung

Die EnhancedSegmentationEngine bietet:

  • Hierarchische Segmentierung: 10 verschiedene Segmenttypen mit Prioritätssystem
  • Qualitätsbewertung: Kohärenz-, Komplexitäts- und Prioritätsscoring
  • Kreuzreferenzen: Automatische Verlinkung zwischen verwandten Segmenten
  • Adaptive Größenanpassung: Inhaltsbasierte Segmentlängenoptimierung
  • Metadaten-Anreicherung: Umfassende Kontextinformationen
# Beispiel: Erweiterte Segmentierung from enhanced_segmentation import EnhancedSegmentationEngine engine = EnhancedSegmentationEngine() segments = engine.segment_with_enhancement(text, metadata) # Ausgabe mit Qualitätsmetriken for segment in segments: print(f"Typ: {segment.segment_type}") print(f"Qualität: {segment.quality_score:.3f}") print(f"Konzepte: {segment.legal_concepts}")

🔄 Datenfluss

Pipeline-Flow: Input → Enhanced Segmentation → Prompt Optimization → Multi-format Output
Schritt Komponente Output
1. Eingabe Data Loader Rohe Rechtstexte
2. Segmentierung EnhancedSegmentationEngine Qualitätsbewertete Segmente
3. Prompt-Gen. OptimizedPromptGenerator Adaptive Prompts
4. Integration OptimizedPipelineIntegrator Training-bereite Daten

⚙️ Konfiguration

Die Pipeline ist vollständig konfigurierbar über optimization_config.json:

{ "prompt_generation": { "max_templates_per_type": 10, "complexity_levels": ["basic", "intermediate", "advanced", "expert"], "enable_domain_adaptation": true }, "segmentation": { "min_segment_length": 50, "max_segment_length": 2000, "quality_threshold": 0.7 }, "rag_training": { "queries_per_segment": 8, "enable_multi_perspective": true } }

🚀 Verwendung

💻 CLI-Interface

Die optimierte Pipeline bietet drei Hauptmodi:

1. Segmentierungsmodus

python optimization_integration.py --mode segmentation \ --input demo_input.jsonl \ --output enhanced_segments.jsonl

2. RAG-Optimierungsmodus

python optimization_integration.py --mode rag \ --input enhanced_segments.jsonl \ --output rag_training_data.jsonl

3. Fine-Tuning Modus

python optimization_integration.py --mode fine-tuning \ --input enhanced_segments.jsonl \ --output fine_tuning_dataset.jsonl

🔧 Programmatische API

from optimized_prompt_generation import OptimizedPromptGenerator from enhanced_segmentation import EnhancedSegmentationEngine # Initialisierung prompt_gen = OptimizedPromptGenerator() seg_engine = EnhancedSegmentationEngine() # Verwendung segments = seg_engine.segment_with_enhancement(text) for segment in segments: prompt = prompt_gen.generate_enhanced_prompt(segment.content) rag_queries = prompt_gen.generate_rag_queries(segment.content)
